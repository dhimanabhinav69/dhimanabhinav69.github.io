Stack Overflow Developer Survey Analysis

An interactive Excel model built to analyse developer roles, programming languages, salary patterns, learning behaviour, and technology adoption using the Stack Overflow Developer Survey dataset.

project objective

Turn raw survey data into clear developer ecosystem insights.

The dashboard helps users explore how developers work, what tools they use, how they learn, and how experience, role, and location influence professional outcomes.

tools used

Excel-based analytics model

Built using Microsoft Excel, Power Query, Pivot Tables, Pivot Charts, slicers, data cleaning, and dashboard design.

Roles developer profile analysis
Skills language and tool trends
Salary experience-based comparison
Learning platform preference insights
